Setup d'un projet

1. Installer Brackets
Extension importantes: Beautify, Git, Icon, Indent guides, Minifier, Php Syntax Hints.
Thème: Brackets Dark.
2. Installer Git
Télécharger sur: http://git-scm.com/downloads
Setup avec identifiant et mot de passe Github
3. Créer un repository sur Github
Suivre ce lien
4. Installer et démarrer MAMP
Ne pas oublier de choisir le dossier contenant les projets à ouvrir via Préférences > Web Server dans MAMP.
Ne pas oublier de permettre l'affichage des erreurs en php via ce lien
5. Télécharger UIkit
UIKit est un framework CSS. Suivre ce lien
5. Sauvegarder un premier fichier index.html ou index.php
Commnencer à coder
6. Faire un premier commit, puis push avec l'extension Git
Vérifier que tout se synchronise

Setup UIKit

// Setup UIKit
<!DOCTYPE html>
<html>

<head>
    <title>MyDashboard - Accueil</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="css/uikit.min.css" />
    <script src="https://code.jquery.com/jquery-3.2.1.min.js"></script>
    <script src="js/uikit.min.js"></script>
    <script src="js/uikit-icons.min.js"></script>
</head>

<body>
</body>
</html>
